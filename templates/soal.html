Demo Sistem Kelola Soal

Tentang web ini

Web ini mendemokan ide tentang web application yang mempunyai tujuan sebagai sistem kelola soal. Baca instruksi dibawah ini untuk memahami lebih lanjut.

  • Pada setiap postingan terdapat 2 tombol. Satu tombol untuk mendownload soal-solusi dalam bentuk pdf dan tombol 'susun' untuk menyusun soal-soal terpilih menjadi dokumen yang tersusun rapi.
  • Pilih soal-soal yang ingin disusun sebagai dokumen. Tampilan "Susunan Sekarang" akan diperbaharui ketika tombol susun ditekan.
  • Tekan tombol konstruksi untuk mendapatkan dokumen pdf dengan tampilan tertata rapi yang berisi soal-soal yang telah disusun.
  • Ditujukan buat para akademisi ketika harus menyusun soal untuk ujian.
  • Klik link video dibawah ini untuk meliat demo.
